Agent Guardrails

Branch scope — Work only in worktrees on the human's personal branch.
Scope guard — Optimizer, multiplayer, mod support, island templates = Phase 2. Do not implement without explicit human approval.
Stack lock — No framework / major library changes after M0 without a decision recorded in docs/.
No silent decisions — Any change spanning >2 files requires a plan before implementation.
Math deferred — Agents scaffold the math engine; humans verify ratio values before merge.
PR hygiene — Every PR must state: what changed, why, and what the human reviewer should check.
